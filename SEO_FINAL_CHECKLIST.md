## üö® COMMON ISSUES & SOLUTIONS

### Issue: Pages Not Indexed
**Solution:**
- Check robots.txt not blocking
- Submit URL in GSC manually
- Check for noindex meta tag
- Wait 1-2 weeks
- Check for soft 404s

### Issue: Low Click-Through Rate
**Solution:**
- Improve title tags (make them click-worthy)
- Better meta descriptions
- Add structured data for snippets
- Improve page rankings
- A/B test titles

### Issue: High Bounce Rate
**Solution:**
- Improve page speed
- Better matching of search intent
- Improve content quality
- Better internal linking
- Improve user experience

### Issue: Crawl Errors
**Solution:**
- Fix server errors (5xx)
- Fix missing pages (404)
- Improve site architecture
- Reduce redirect chains
- Optimize robots.txt
